PlayerData

Name Type Necessity Description Example
user UserData Required The information about user. "user": { ... }
userInvitesMe Boolean Required Indicates if that user has invited current user. "userInvitesMe": true
userIsInvitedByMe Boolean Required Indicates if that current user has invited that user. "userIsInvitedByMe": false
