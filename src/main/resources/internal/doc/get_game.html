Chess REST API Documentation

GET /games/:id

Return the game information.

Request

GET http://{host}:{port}/api/games/{gameId}?version={version}

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request parameters

Name Type Necessity Description Example
gameId String Required The unique identifier of the game. "314d2352-402d-4da9-a8c7-8f6aa446687e"
version Integer Optional The version of player list. 42

Example of request

GET http://{host}:{port}/api/games/314d2352-402d-4da9-a8c7-8f6aa446687e?version=42
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Response

Status Code Description
200 OK   The game information is returned successfully.
204 No Content   The game version is not changed. The content is not returned.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
404 Not Found The requested game is not found.

See Error Codes & Responses

Response body

Response body contains GameData

blackPlayer UserData Required The information about black player. "blackPlayer": { ... }
