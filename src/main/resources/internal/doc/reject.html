PUT /invitations/reject

Reject invitation.

Request

PUT http://{host}:{port}/api/invitations/reject?id={id}

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request parameters

Name Type Necessity Description Example
id String Required The unique identifier of invitation. baabfa7d-74f5-48ab-bf14-f1fdb62d2147

Example of request

PUT http://{host}:{port}/api/invitations/reject?id=baabfa7d-74f5-48ab-bf14-f1fdb62d2147
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Response

Status Code Description
200 OK   The operation is performed successfully.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden Access denied. Only invitee can reject the invitation.
404 Not Found The invitation not found.
409 Confict Incorrect invitation status.

See Error Codes & Responses

Response body contains InvitationData
