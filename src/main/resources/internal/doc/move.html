Chess REST API Documentation

POST /games/:id/move

Make movement.

Request

POST http://{host}:{port}/api/games/{gameId}/move?src={src}&dst={dst}&promoted={promoted}

Request header

Name Description Example
Authorization The token to authorize the user. Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Request parameters

Name Type Necessity Description Example
gameId String Required The unique identifier of game to make movement. baabfa7d-74f5-48ab-bf14-f1fdb62d2147
src String Required Source square of movement. e2
dst String Required Destination square of the movement. e4
promoted String Optional The piece kind to promoted into. Valid only for promotion. Q

Example of request

POST http://{host}:{port}/api/games/baabfa7d-74f5-48ab-bf14-f1fdb62d2147/move?src=e2&dst=e4
Authorization: 15d07e2d-6e0d-470f-8de9-3a8ce0afa934

Response

Status Code Description
200 OK   The operation is performed successfully.
400 Bad Request Invalid movement.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden Access denied. Only player may make movements.
404 Not Found The game not found.
