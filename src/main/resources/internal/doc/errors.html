Error Codes & Responses

Response

Status Code Description
200 OK   The operation is performed successfully.
204 No Content   The server successfully processed the request and is not returning any content.
400 Bad Request The request content validation is failed.
401 Unauthorized The resource requires authentication, which was not supplied with the request.
The supplied authentication is invalid.
403 Forbidden The requested resource could not be found but may be available again in the future.
404 Not Found The requested resource could not be found but may be available again in the future.
The requested user is not found.
The requested game is not found.
409 Conflict User already exists.
User can't invite self.
Incorrect invitation status.
500 Internal Server Error There was an internal server error.
The request processing is timed out.

Response body

Name Type Necessity Description Example
status Int Required HTTP status code. "status": 404
code String Required The error code. "code": "USER_NOT_FOUND"
message String Required The error message. "message": "User not found."

Example of response body

{
    "status": 404,
    "code": "USER_NOT_FOUND",
    "message": "User not found."
}
