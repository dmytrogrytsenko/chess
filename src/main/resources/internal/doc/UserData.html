Chess REST API Documentation

UserData

Name Type Necessity Description Example
id String Required The unique identifier of the user. "id": "baabfa7d-74f5-48ab-bf14-f1fdb62d2147"
name String Required The user name. "name": "janedoe"
displayName String Optional The display name of the user. "displayName": "Jane Doe"
createdAt DateTime Required The user creation date and time in UTC. "createdAt": "2016-04-30T10:34:07.341Z"
