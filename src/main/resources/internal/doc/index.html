Chess REST API Documentation

Error Codes & Responses

User operations

Method Resource Description
POST /register Register new user.
POST /login Login the user.
POST /logout Logout the user.
GET /profile Return the user profile information.

Player operations

Method Resource Description
GET /players Return the online player list.
POST /invite Invite other user to play game.
PUT /invitations/cancel Cancel own invitation.
PUT /invitations/reject Reject invitation.
PUT /invitations/accept Accept invitation.

Game operations

Method Resource Description
GET /games Return the player's game list.
GET /games/:id Return game information.
