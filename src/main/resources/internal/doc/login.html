Chess REST API Documentation

POST /login

Login the user.

Request

POST http://{host}:{port}/login

Request body

Name Type Necessity Description Example
name String Required The user name. "name": "janedoe"
password String Required The user password. "password": "Ae71bags&534"

Example of request

POST http://{host}:{port}/login
{
    "name": "janedoe",
    "password": "Ae71bags&534"
}

Response

Status Code Description
200 OK   User is logged in successfully.
400 Bad Request

The request content validation is failed:

  • Required member 'name' is empty.
  • Required member 'password' is empty.
    •
    401 Unauthorized The supplied authentication is invalid.

    See Error Codes & Responses

    Response body

    Name Type Necessity Description Example
    token String Required The authorization token. "token": "baabfa7d-74f5-48ab-bf14-f1fdb62d2147"
    user UserData Required The user information. "user": { ... }
